# 3.20 - DOS function 43h - Change File Attributes
| **In:** | **AH** = 43h<br> **AL** = function code<br> **ECX** = desired file attributes<br> **DS:EDX** = pointer to ASCIIZ file name |
|---|---|
| **Out:** | <br><br>if successful:<br> **CF** clear<br> **ECX** = current attributes <br><br>if failed:<br> **CF** set<br> **EAX** = DOS error code |
