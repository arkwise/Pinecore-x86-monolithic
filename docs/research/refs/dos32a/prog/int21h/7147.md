# 3.39 - DOS function 7147h - Win95 LFN Get Current Directory
| **In:** | **AX** = 7147h<br> **DL** = drive number<br> **DS:ESI** = pointer to buffer for ASCIIZ long directory name |
|---|---|
| **Out:** | <br><br>if successful:<br> **CF** clear <br><br>if failed:<br> **CF** set<br> **EAX** = DOS error code |
