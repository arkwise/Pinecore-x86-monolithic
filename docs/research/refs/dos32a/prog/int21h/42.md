# 3.19 - DOS function 42h - Set Current File Position
| **In:** | <br><br>**AH** = 42h<br> **AL** = origin of move: <br><br>00h = start<br> 01h = current<br> 02h = end<br> <br><br>**ECX:EDX** = file position |
|---|---|
| **Out:** | <br><br>if successful:<br> **CF** clear<br> **EDX:EAX** = current file position <br><br>if failed:<br> **CF** set<br> **EAX** = DOS error code |
