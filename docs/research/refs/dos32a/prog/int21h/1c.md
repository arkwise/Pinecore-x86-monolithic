# 3.04 - DOS function 1Ch - Get Specific Drive Info
| **In:** | **AH** = 1Ch<br> **DL** = drive number |
|---|---|
| **Out:** | <br><br>if successful:<br> **AL** = sectors per cluster<br> **ECX** = bytes per sector<br> **EDX** = total number of clusters on drive<br> **DS:EBX** = pointer to media ID byte <br><br>if failed:<br> **AL** = 0FFh (invalid drive in DL) |

**Notes: **

a) The selector returned in DS register will be zero selector, ie readable/writeable data selector with Base=0. This is an internal to DOS/32 Advanced selector which may NOT be freed or modified under any circumstances.
