# 3.46 - DOS function 0FF89h - DOS/32 Advanced Get Configuration Info
| **In:** | **AX** = 0FF89h |
|---|---|
| **Out:** | **EAX** = 49443332h ('ID32') (if the function is supported)<br> **EBX** = DOS/32 Advanced version number<br> **ECX** = DOS/32 Advanced Low Buffer size (in bytes)<br> **EDX** = DOS/32 Advanced Configuration bits<br> **ESI** = pointer to DOS/32 Advanced Configuration Header "ID32"<br> **FS** = selector that can be used to access Configuration Header "ID32" |

**Notes: **

a) This call is specific to DOS/32 Advanced DOS Extender only, and is not supported by standard DOS.
