# 3.22 - DOS function 48h - Allocate DOS Memory
| **In:** | **AH** = 48h<br> **EBX** = paragraphs to allocate |
|---|---|
| **Out:** | <br><br>if successful:<br> **CF** clear<br> **EAX** = selector of allocated memory block <br><br>if failed:<br> **CF** set<br> **EAX** = DOS error code<br> **EBX** = maximum paragraphs available |
