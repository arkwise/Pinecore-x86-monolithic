# 3.41 - DOS function 7160h - Win95 LFN Get File True Name
| **In:** | <br><br>**AX** = 7160h<br> **CL** = subfunction: <br><br>00h - canonicalize path<br> 01h - get short (8.3) filename for file<br> 02h - get canonical long filename or path <br><br>**CH** = SUBST expansion flag<br> **DS:ESI** = pointer to ASCIIZ filename or path<br> **ES:EDI** = 128/261-byte buffer for canonicalized name |
|---|---|
| **Out:** | <br><br>if successful:<br> **CF** clear<br> buffer filled with canonicalized file name <br><br>if failed:<br> **CF** set<br> **EAX** = DOS error code |

**Notes:**

a) Subfunctions 00h and 02h will require the buffer size to be at least 261 bytes, subfunction 01h will require the buffer size to be at least 128 bytes.
