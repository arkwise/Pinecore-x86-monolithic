# 3.17 - DOS function 40h - Write to File
| **In:** | **AH** = 40h<br> **EBX** = file handle<br> **ECX** = number of bytes to write (size)<br> **DS:EDX** = pointer to buffer to write from (addr) |
|---|---|
| **Out:** | <br><br>if successful:<br> **CF** clear<br> **EAX** = number of bytes written <br><br>if failed:<br> **CF** set<br> **EAX** = DOS error code |
