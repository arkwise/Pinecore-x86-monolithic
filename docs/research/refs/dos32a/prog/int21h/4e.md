# 3.27 - DOS function 4Eh - Find First File
| **In:** | **AH** = 4Eh<br> **ECX** = file attributes<br> **DS:EDX** = pointer to ASCIIZ file name |
|---|---|
| **Out:** | <br><br>if successful:<br> **CF** clear<br> DTA is filled with file information <br><br>if failed:<br> **CF** set<br> **EAX** = DOS error code |
