# 3.42 - DOS function 716Ch - Win95 LFN Create/Open File
| **In:** | **AX** = 716Ch<br> **EBX** = access mode and sharing flags<br> **ECX** = file attributes<br> **EDX** = action<br> **EDI** = alias hint<br> **DS:ESI** = pointer to ASCIIZ filename |
|---|---|
| **Out:** | <br><br>if successful:<br> **CF** clear<br> **EAX** = file handle<br> **ECX** = action taken: <br><br>01h - file opened<br> 02h - file created<br> 03h - file replaced <br><br>if failed:<br> **CF** set<br> **EAX** = DOS error code |
