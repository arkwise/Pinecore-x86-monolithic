# 3.16 - DOS function 3Fh - Read from File
| **In:** | **AH** = 3Fh<br> **EBX** = file handle<br> **ECX** = number of bytes to read (size)<br> **DS:EDX** = pointer to buffer to read to (addr) |
|---|---|
| **Out:** | <br><br>if successful:<br> **CF** clear<br> **EAX** = number of bytes read <br><br>if failed:<br> **CF** set<br> **EAX** = DOS error code |
