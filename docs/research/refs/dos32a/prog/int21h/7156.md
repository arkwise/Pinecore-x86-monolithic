# 3.40 - DOS function 7156h - Win95 LFN Rename File
| **In:** | **AX** = 7156h<br> **DS:EDX** = pointer to ASCIIZ old long file or directory name<br> **ES:EDI** = pointer to ASCIIZ new long file or directory name |
|---|---|
| **Out:** | <br><br>if successful:<br> **CF** clear <br><br>if failed:<br> **CF** set<br> **EAX** = DOS error code |
