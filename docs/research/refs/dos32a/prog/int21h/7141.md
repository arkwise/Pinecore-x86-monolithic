# 3.37 - DOS function 7141h - Win95 LFN Delete File
| **In:** | **AX** = 7141h<br> **CL** = search attributes<br> **CH** = must-match attributes<br> **SI** = wildcards and attributes flag<br> **DS:EDX** = pointer to ASCIIZ long name of file to delete |
|---|---|
| **Out:** | <br><br>if successful:<br> **CF** clear <br><br>if failed:<br> **CF** set<br> **EAX** = DOS error code |
