# 3.24 - DOS function 4Ah - Resize DOS Memory
| **In:** | **AH** = 4Ah<br> **EBX** = new memory block size in paragraphs<br> **ES** = selector of previously allocated memory block |
|---|---|
| **Out:** | <br><br>if successful:<br> **CF** clear <br><br>if failed:<br> **CF** set<br> **EAX** = DOS error code<br> **EBX** = maximum paragraphs available for specific memory block |
