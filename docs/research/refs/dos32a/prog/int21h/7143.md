# 3.38 - DOS function 7143h - Win95 LFN Get/Set File Attributes
| **In:** | <br><br>**AX** = 7143h<br> **BL** = action <br><br>00h - get attributes<br> 01h - set attributes <br><br>if **BL** = 01h, **ECX** = file attributes<br> **DS:EDX** = pointer to ASCIIZ long filename |
|---|---|
| **Out:** | <br><br>if successful:<br> **CF** clear<br> if **BL**=00h, **ECX** = file attributes <br><br>if failed:<br> **CF** set<br> **EAX** = DOS error code |

**Notes:**

a) Only the first two subfunctions (actions) are supported by this function, BL=00h and BL=01h.
