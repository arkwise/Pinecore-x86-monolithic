# 3.21 - DOS function 47h - Get Directory Path
| **In:** | **AH** = 47h<br> **DL** = drive number<br> **DS:ESI** = pointer to buffer for path |
|---|---|
| **Out:** | <br><br>if successful:<br> **CF** clear<br> buffer filled with the path name <br><br>if failed:<br> **CF** set<br> **EAX** = DOS error code |
