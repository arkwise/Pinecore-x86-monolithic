# 3.50 - DOS function 0FF92h - DOS/32 Advanced Free High Memory Block
| **In:** | **AX** = 0FF92h<br> **ESI** = handle of previously allocated memory block |
|---|---|
| **Out:** | <br><br>if successful:<br> **CF** clear <br><br>if failed:<br> **CF** set |

**Notes: **

a) This call is specific to DOS/32 Advanced DOS Extender only, and is not supported by standard DOS.

b) This function is equal to DPMI function 0502h except that it uses 32-bit register instead of 16:16-bit pair of registers.
