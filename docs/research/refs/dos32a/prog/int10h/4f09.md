# 4.06 - VBE function 4F09h - Load/Unload Palette Data
| **In:** | <br><br>**AX** = 4F09h<br> **BL** = subfunction: <br><br>00h - set palette data<br> 01h - get palette data<br> 02h - set 2nd palette data<br> 03h - get 2nd palette data<br> 80h - set palette data during VR <br><br>**ECX** = number of palette registers to update<br> **EDX** = first palette register to update<br> **ES:EDI** = pointer to buffer that contains palette data |
|---|---|
| **Out:** | **AL** = 4Fh (if function supported by VESA BIOS)<br> **AH** = 00h (if function was successful) |
