# 4.04 - VBE function 4F01h - Get SuperVGA Mode Information
| **In:** | **AX** = 4F01h<br> **ES:EDI** = pointer to 256 byte buffer |
|---|---|
| **Out:** | **AL** = 4Fh (if function supported by VESA BIOS)<br> **AH** = 00h (if function was successful)<br> **ES:EDI** = pointer to 256 byte buffer filled with VESA mode information |
