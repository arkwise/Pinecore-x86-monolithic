# 4.05 - VBE function 4F04h - Save/Restore SuperVGA State
| **In:** | <br><br>**AX** = 4F04h<br> **DL** = subfunction: <br><br>00h - get state buffer size<br> **Returns: EBX** = number of 64-byte blocks needed<br> 01h - save video states<br> 02h - restore video states <br><br>**ECX** = states to save/restore: <br><br>bit 0 - SVGA hardware state<br> bit 1 - SVGA BIOS data state<br> bit 2 - SVGA DAC state<br> bit 3 - SVGA state <br><br>**ES:EBX** = pointer to save/restore buffer |
|---|---|
| **Out:** | **AL** = 4Fh (if function supported by VESA BIOS)<br> **AH** = 00h (if function was successful) |

**Notes: **

a) For subfunctions 01h and 02h, the DOS Extender will internally call subfunction 00h to extract the information about size of data that is to be copied.
