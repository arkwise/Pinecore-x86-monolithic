# 2.57 - API function 04h - Get Access to Real Mode Virtual Stacks
| **In:** | **AL** = 04h |
|---|---|
| **Out:** | **BX** = writeable data selector<br> **ECX** = size of one real mode virtual stack<br> **EDX** = pointer to the current top of real mode virtual stack<br> **ESI** = pointer to base of real mode virtual stacks |

**Notes:**

a) Selector returned in BX is a writeable data selector that should be used to access needed variables/addresses.
