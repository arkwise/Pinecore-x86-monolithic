# 2.33 - DPMI function 0502h - Free Memory Block
Frees a memory block that was previously allocated with the Allocate Memory Block DPMI function 0501h.

| **In:** | **AX** = 0502h<br> **SI:DI** = memory block handle |
|---|---|
| **Out:** | <br><br>if successful:<br> **CF** clear <br><br>if failed:<br> **CF** set<br> **AX** = error code <br><br>**8023h** - invalid handle |

**Notes: **

a) No descriptors are freed by this call. It is the client's responsibility to free any descriptors that it previously allocated to map the memory block. Descriptors should be freed before linear memory blocks.
