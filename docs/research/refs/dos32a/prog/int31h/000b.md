# 2.10 - DPMI function 000Bh - Get Descriptor
Copies the descriptor table entry for the specified selector into an 8-byte buffer.

| **In:** | **AX** = 000Bh<br> **BX** = selector<br> **ES:EDI** = selector:offset of 8-byte buffer |
|---|---|
| **Out:** | <br><br>if successful:<br> **CF** clear<br> the buffer pointed to by ES:EDI contains descriptor <br><br>if failed<br> **CF** set<br> **AX** = error code <br><br>**8022h** - invalid selector |
