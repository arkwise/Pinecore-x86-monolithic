# 2.22 - DPMI function 0205h - Set Protected Mode Interrupt Vector
Sets the address of protected mode handler for the specified interrupt into the interrupt vector.

| **In:** | **AX** = 0205h<br> **BL** = interrupt number<br> **CX:EDX **= selector:offset of exception handler |
|---|---|
| **Out:** | <br><br>if successful:<br> **CF** clear <br><br>if failed:<br> **CF** set<br> **AX** = error code <br><br>**8022h** - invalid selector |

**Notes: **

a) The value passed in CX should be a valid protected mode code selector, not a real mode segment address.
