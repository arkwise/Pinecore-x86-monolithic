# 2.41 - DPMI function 0702h - Mark Page as Demand Paging Candidate
Notifies the DPMI host that a range of pages may be placed at the head of the page-out candidate list, forcing these pages to be replaced ahead of other pages even if the memory has been accessed recently. The contents of the pages will be preserved.

| **In:** | **AX** = 0702h<br> **BX:CX** = starting linear address of pages to mark as paging candidates<br> **SI:DI** = size of region to mark (bytes) |
|---|---|
| **Out:** | <br><br>if successful:<br> **CF** clear <br><br>if failed:<br> **CF** set<br> **AX** = error code <br><br>**8025h** - invalid linear address (range unallocated) |

**Notes: **

a) DOS/32 Advanced DPMI will perform no action on this function and will always return with carry flag clear.
