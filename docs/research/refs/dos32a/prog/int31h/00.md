# 2.53 - API function 00h - Get Access to GDT and IDT
| **In:** | **AL** = 00h |
|---|---|
| **Out:** | **BX** = writeable data selector<br> **ECX** = GDT limit<br> **EDX** = IDT limit<br> **ESI** = pointer to GDT<br> **EDI** = pointer to IDT |

**Notes: **

a) Selector returned in BX is a writeable data selector that should be used to access needed variables/addresses.
