# 2.12 - DPMI function 000Eh - Get Multiple Descriptors
Copies one or more descriptor table entries into a client buffer.

**Length**  **Contents**   00h  2 Selector #1 (set by client)   02h  8  Descriptor #1 (returned by host)   0Ah  2  Selector #2 (set by client)   0Ch  8  Descriptor #2 (returned by host)   . . . . . .  . . .

| **In:** | <br><br>**AX** = 000Eh<br> **CX** = number of descriptors to copy<br> **ES:EDI**= selector:offset of a buffer in the following format: <br><br> **Offset** |
|---|---|
| **Out:** | <br><br>if successful:<br> **CF** clear<br> the buffer contains copies of the descriptors for the specified selectors <br><br>if failed:<br> **CF** set<br> **AX** = error code<br> **CX** = number of descriptors successfully copied <br><br>**8022h** - invalid selector |

**Notes: **

a) If an error occurs because of an invalid selector or descriptor, the function returns the number of descriptors which were successfully copied in CX. All of the descriptors which were copied prior to the one that failed are valid.
