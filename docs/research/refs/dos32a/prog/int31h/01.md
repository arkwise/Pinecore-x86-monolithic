# 2.54 - API function 01h - Get Access to Page Tables
| **In:** | **AL** = 01h |
|---|---|
| **Out:** | **BX** = writeable data selector<br> **ECX** = number of allocated page tables<br> **EDX** = number of allocated page tables for physical address mapping<br> **ESI** = pointer to 0-th page table<br> **EDI** = pointer to first page table for physical address mapping |

**Notes: **

a) Selector returned in BX is a writeable data selector that should be used to access needed variables/addresses.

b) This function will return the correct values under VCPI only.
