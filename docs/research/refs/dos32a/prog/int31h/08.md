# 2.61 - API function 08h - Set Critical Handler Entry Point
| **In:** | **AL** = 08h<br> **CX:DX** = 16:16 selector:offset pointer to critical handler routine |
|---|---|
| **Out:** | **-** |

**Notes: **

a) The selector and offset values passed to this function must be 16-bit values, ie the critical handler must reside in a 16-bit code segment.
