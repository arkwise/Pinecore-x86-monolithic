# 2.17 - DPMI function 0200h - Get Real Mode Interrupt Vector
Returns the contents of the current virtual machine's real mode interrupt vector for the specified interrupt.

| **In:** | **AX** = 0200h<br> **BL** = interrupt number |
|---|---|
| **Out:** | **CF** clear<br> **CX:DX** = segment:offset of real mode interrupt handler |

**Notes: **

a) The value returned in CX is a real mode segment address, not a selector. Attempts to place this value into a segment register in protected mode may cause a General Protection (GP) fault exception.
