# 2.58 - API function 05h - Get Access to Protected Mode Virtual Stacks
| **In:** | **AL** = 05h |
|---|---|
| **Out:** | **BX** = writeable data selector<br> **ECX** = size of one protected mode virtual stack<br> **EDX** = pointer to the current top of protected mode virtual stack<br> **ESI** = pointer to base of protected mode virtual stacks |

**Notes: **

a) Selector returned in BX is a writeable data selector that should be used to access needed variables/addresses.
