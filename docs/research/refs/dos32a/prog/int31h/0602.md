# 2.38 - DPMI function 0602h - Mark Real Mode Region as Pageable
Advises the DPMI host that the specified memory below the 1 MB boundary may be paged to disk.

| **In:** | **AX** = 0602h<br> **BX:CX** = starting linear address of memory to mark as pageable<br> **SI:DI** = size of region to be marked (bytes) |
|---|---|
| **Out:** | <br><br>if successful:<br> **CF** clear <br><br>if failed:<br> **CF** set<br> **AX** = error code <br><br>**8002h** - invalid state (region already marked as pageable)<br> **8025h** - invalid linear address (region is above 1MB boundary) |

**Notes:**

a) As DOS/32 Advanced does not support virtual memory this function is implemented for compatibility reasons only.
