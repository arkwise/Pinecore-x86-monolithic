# 2.35 - DPMI function 050Ah - Get Memory Block Size and Base
Returns the size of a memory block that was previously allocated with DPMI function 0501h.

| **In:** | **AX** = 050Ah<br> **SI:DI** = memory block handle |
|---|---|
| **Out:** | <br><br>if successful:<br> **CF** clear<br> **SI:DI** = size of memory block (bytes)<br> **BX:CX** = base address of memory block <br><br>if failed:<br> **CF** set<br> **AX** = error code <br><br>**8023h** - invalid handle |
