# 2.56 - API function 03h - Get Access to Extended Memory Blocks
| **In:** | **AL** = 03h |
|---|---|
| **Out:** | **BX** = writeable data selector<br> **ECX** = total size of allocated extended memory<br> **EDX** = pointer to allocated extended memory (first byte)<br> **ESI** = pointer to top of allocated extended memory (last byte) |

**Notes: **

a) Selector returned in BX is a writeable data selector that should be used to access needed variables/addresses.
