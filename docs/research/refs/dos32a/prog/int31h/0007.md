# 2.06 - DPMI function 0007h - Set Segment Base Address
Sets the 32-bit linear base address field in the descriptor table for the specified segment.

| **In:** | **AX** = 0007h<br> **BX** = selector<br> **CX:DX** = 32-bit linear base address of segment |
|---|---|
| **Out:** | <br><br>if successful:<br> **CF** clear <br><br>if failed:<br> **CF** set<br> **AX** = error code <br><br>**8022h** - invalid selector<br> **8025h** - invalid linear address (changing the base would cause the descriptor to reference a linear address range outside that allowed for DPMI clients) |

**Notes: **

a) DOS/32 Advanced DPMI and DPMI 1.0 hosts will automatically reload any segment register which contains the selector specified in register BX.
