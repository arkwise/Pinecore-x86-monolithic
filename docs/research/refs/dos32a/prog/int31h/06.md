# 2.59 - API function 06h - Get DOS/32 Advanced DPMI Kernel Selectors
| **In:** | **AL** = 06h |
|---|---|
| **Out:** | **BX** = DOS/32 Advanced DPMI Kernel CODE selector<br> **CX** = DOS/32 Advanced DPMI Kernel DATA selector<br> **DX** = DOS/32 Advanced DPMI Kernel ZERO selector<br> **SI** = DOS/32 Advanced DPMI Kernel CODE segment (real-mode)<br> **DI** = DOS/32 Advanced DOS Extender Client CODE selector |

**Notes: **

a) Selectors returned by this function should NOT be modified or freed by the user.
