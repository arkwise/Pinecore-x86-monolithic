# 5.07 - Mouse function 0019h - Return User Alternate Interrupt Handler
| **In:** | **AX** = 0019h<br> **ECX** = call mask |
|---|---|
| **Out:** | **ECX** = call mask<br> **BX:EDX** = user interrupt vector |

**Notes:**

a) This function will simply return the address of the routine defined by functions 000Ch, 0014h and 0018h.
