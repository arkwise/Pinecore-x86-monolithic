# 5.08 - Mouse function 0020h - Enable Mouse Driver
| **In:** | **AX** = 0020h |
|---|---|
| **Out:** | **AX** = 0020h on error<br> **AX** = 0FFFFh on success |

**Notes:**

a) This function will always return AX=0FFFFh to be compatible with DOS/4GW DOS Extender.

b) The protected mode program should not rely on error codes reported by this function in AX register since different mouse drivers will return different values on error.
