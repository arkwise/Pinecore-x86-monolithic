# 2.08 - DOS/32 Advanced D32A Library function - *d32a_get_windows_version()
**Synopsys:**

*

#include <d32a.h>
int d32a_get_windows_version(void);

*

**Description:**

This function returns the version number of Microsoft Windows(R) Operating System.

**Returns:**

Version of MS-Windows(R). For example under Windows 95 this function will return 0x0004. When the application is not running under Microsoft Windows (see *d32a_detect_windows* function) this function will return -1.
