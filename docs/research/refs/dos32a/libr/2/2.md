# 2.02 - DOS/32 Advanced D32A Library function - *d32a_detect_adpmi()
**Synopsys:**

*

#include <d32a.h>
int d32a_detect_adpmi(void);

*

**Description:**

Detects DOS/32 Advanced built-in DPMI server.

**Returns:**

**TRUE** if DOS/32 Advanced built-in DPMI is present (the case when running under Clean, XMS or VCPI), **FALSE** otherwise (when the application is running under an external DPMI host).
