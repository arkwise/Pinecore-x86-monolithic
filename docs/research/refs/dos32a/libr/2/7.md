# 2.07 - DOS/32 Advanced D32A Library function - *d32a_get_dos_version()
**Synopsys:**

*

#include <d32a.h>
int d32a_get_dos_version(void);

*

**Description:**

This function returns the version number of DOS Operating System.

**Returns:**

Version of DOS. For example under DOS version 7.10 this function will return 0x070A.
