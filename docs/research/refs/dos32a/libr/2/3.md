# 2.03 - DOS/32 Advanced D32A Library function - *d32a_detect_system_software()
**Synopsys:**

*

#include <d32a.h>
int d32a_detect_system_software(void);

*

**Description:**

Finds out the type of the present System Software, either Clean (also known as "Raw" or INT 15h), XMS, VCPI or DPMI.

**Returns:**

When successful, this function returns the type of the System Software as defined below:

**D32A_SYSTEM_CLEAN** 0
**D32A_SYSTEM_XMS** 1
**D32A_SYSTEM_VCPI** 2
**D32A_SYSTEM_DPMI** 3
**D32A_SYSTEM_UNKNOWN** -1

If failed (when not running under DOS/32 Advanced DOS Extender, and the type of the System Software cannot be determined) this function will return -1.
