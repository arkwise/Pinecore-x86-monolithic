# 2.09 - DOS/32 Advanced D32A Library function - *d32a_get_library_version()
**Synopsys:**

*

#include <d32a.h>
int d32a_get_library_version(void);

*

**Description:**

This function returns the version number of D32A C/C++ Run-Time Library which has been statically linked into the application.

**Returns:**

Version of D32A C/C++ Run-Time Library. For example in a hypothetical version 5.43 the returned value will be 0x052B.
