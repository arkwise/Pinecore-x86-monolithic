# 4.06 - DOS/32 Advanced D32A Library function - *d32a_getfreemem()
**Synopsys:**

*

#include <d32a.h>
unsigned long d32a_getfreemem(void);

*

**Description:**

This function allows you to determine how much free Extended (High or DPMI) memory is currently available for application's use. The value returned by this function should be considered as advisory when the application is running in a multitasking environment, as other programs may allocate some memory after this function has been called.

**Returns:**

The amount of available free Extended (High) memory in bytes.
