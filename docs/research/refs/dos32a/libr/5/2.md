# 5.02 - DOS/32 Advanced D32A Library function - *d32a_decode_data()
**Synopsys:**

*

#include <d32a.h>
unsigned long d32a_decode_data(char *Src, char *Dest, unsigned long Size);

*

**Description:**

This function decompresses **Size** bytes of data from the **Src** buffer, and places the produced result into the **Dest** buffer.

**Returns:**

The size of the produced (decompressed) data that was placed into the **Dest** buffer in bytes.
