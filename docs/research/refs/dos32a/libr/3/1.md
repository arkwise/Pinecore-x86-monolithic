# 3.01 - DOS/32 Advanced D32A Library function - *d32a_kbhit()
**Synopsys:**

*

#include <d32a.h>
int d32a_kbhit(void);

*

**Description:**

The *d32a_kbhit* function tests whether or not a keystroke is currently available. The *getch* function of WATCOM C/C++ standard Run-Time library can be used to retrieve the character from the buffer.

**Returns:**

**TRUE** when a keystroke is available, **FALSE** otherwise.
