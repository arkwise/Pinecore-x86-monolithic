# 9.0 - DOS/32 Advanced - Redistributable Components
For information about DOS/32 Advanced DOS Extender Liberty Edition usage and distribution refer to [DOS/32 Advanced DOS Extender Liberty Edition Software License](0C.HTM).
**Note:**
When distributing DOS/32 Advanced DOS Extender Liberty Edition as a whole, it is strongly adviced (although not required) to ship the package in installable form as opposed to simply zipping DOS/32A directory structure after the installation and distributing the Zip file instead.
