# 1.0 - DOS/32 Advanced - Overview
DOS/32 Advanced DOS Extender is built up of three parts, that can be seen as three different programs: the DOS Extender itself which extends software interrupts INT 10h, INT 21h and INT 33h, a built-in Advanced DPMI host v0.9, ADPMI, which provides your protected mode program with a subset of DPMI functions via software interrupt INT 31h, and a Loader which loads your protected mode application when the protected mode is entered. In the following three sections we will discuss the actions taken by DOS/32 Advanced on three different stages: the startup, loading an application and exit to DOS.
