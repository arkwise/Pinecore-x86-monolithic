# NE2000 ISA Ethernet NIC (DP8390)

> The most common ISA NIC. Based on National Semiconductor DP8390. NE2000 compatible cards dominated the DOS networking era.

**Date:** 2026-05-02
**Status:** Complete — register-level reference for implementation
**Source:** National Semiconductor DP8390 datasheet; Novell NE2000 reference

---

## Findings

### Hardware Overview

- ISA 8/16-bit card, typically at I/O 0x300 (configurable: 0x280, 0x300, 0x320, 0x340)
- IRQ 3, 5, 9, or 10 (jumper-selectable)
- On-board 16KB or 32KB packet buffer SRAM
- No DMA — uses PIO (Remote DMA via the NIC's internal DMA engine)
- Register pages: 3 pages selected via bits 6-7 of Command Register

### Register Map

#### Command Register (offset 0x00, all pages)

| Bit | Name | Purpose |
|-----|------|---------|
| 0 | STP | Stop (1=stop NIC) |
| 1 | STA | Start (1=start NIC) |
| 2 | TXP | Transmit Packet (1=initiate send) |
| 5-3 | RD | Remote DMA: 001=read, 010=write, 100=abort |
| 7-6 | PS | Page Select: 00=page0, 01=page1, 10=page2 |

#### Page 0 Registers (read)

| Offset | Name | Purpose |
|--------|------|---------|
| 0x01 | CLDA0 | Current Local DMA Address low |
| 0x02 | CLDA1 | Current Local DMA Address high |
| 0x03 | BNRY | Boundary Pointer (Rx ring read position) |
| 0x04 | TSR | Transmit Status Register |
| 0x07 | ISR | Interrupt Status Register |
| 0x0C | RSR | Receive Status Register |
| 0x0D | CNTR0 | Alignment error counter |
| 0x0E | CNTR1 | CRC error counter |
| 0x0F | CNTR2 | Missed packet counter |

#### Page 0 Registers (write)

| Offset | Name | Purpose |
|--------|------|---------|
| 0x01 | PSTART | Page Start (Rx ring start page) |
| 0x02 | PSTOP | Page Stop (Rx ring end page) |
| 0x03 | BNRY | Boundary Pointer |
| 0x04 | TPSR | Transmit Page Start Register |
| 0x05 | TBCR0 | Transmit Byte Count low |
| 0x06 | TBCR1 | Transmit Byte Count high |
| 0x07 | ISR | Interrupt Status (write 1 to clear) |
| 0x08 | RSAR0 | Remote Start Address low (DMA source/dest) |
| 0x09 | RSAR1 | Remote Start Address high |
| 0x0A | RBCR0 | Remote Byte Count low |
| 0x0B | RBCR1 | Remote Byte Count high |
| 0x0C | RCR | Receive Configuration |
| 0x0D | TCR | Transmit Configuration |
| 0x0E | DCR | Data Configuration |
| 0x0F | IMR | Interrupt Mask Register |

#### Page 1 Registers

| Offset | Name | Purpose |
|--------|------|---------|
| 0x01-0x06 | PAR0-5 | Physical Address (MAC, 6 bytes) |
| 0x07 | CURR | Current Page (Rx ring write pointer) |
| 0x08-0x0F | MAR0-7 | Multicast Address hash table |

#### Remote DMA Port

| Offset | Size | Purpose |
|--------|------|---------|
| 0x10-0x17 | 16-bit | Remote DMA data port (read/write packet data) |
| 0x18 | 8-bit | Reset port (write any value) |

### ISR Bits

| Bit | Name | Meaning |
|-----|------|---------|
| 0 | PRX | Packet Received OK |
| 1 | PTX | Packet Transmitted OK |
| 2 | RXE | Receive Error |
| 3 | TXE | Transmit Error |
| 4 | OVW | Rx Ring Overflow |
| 5 | CNT | Tally Counter Overflow |
| 6 | RDC | Remote DMA Complete |
| 7 | RST | Reset Status |

### DCR (Data Configuration)

| Bit | Name | Purpose |
|-----|------|---------|
| 0 | WTS | 1=16-bit (word) transfers, 0=8-bit |
| 1 | BOS | Byte Order (0=little-endian) |
| 5 | FT1 | FIFO threshold |

### NIC Memory Layout (16KB card)

```
Page 0x40-0x45: Transmit buffer (6 pages = 1536 bytes, fits one Ethernet frame)
Page 0x46-0x7F: Receive ring buffer (58 pages = ~14KB)
```
Each page = 256 bytes.

### Initialization Sequence

```
1.  outb(io+0x00, 0x21)    -- CR: stop, abort DMA, page 0
2.  outb(io+0x0E, 0x49)    -- DCR: word mode, FIFO=2, normal
3.  outb(io+0x0A, 0x00)    -- RBCR0: clear remote byte count
4.  outb(io+0x0B, 0x00)    -- RBCR1
5.  outb(io+0x0C, 0x20)    -- RCR: monitor mode (during init)
6.  outb(io+0x0D, 0x02)    -- TCR: internal loopback (during init)
7.  outb(io+0x01, 0x46)    -- PSTART: Rx ring starts at page 0x46
8.  outb(io+0x02, 0x80)    -- PSTOP: Rx ring ends at page 0x80
9.  outb(io+0x03, 0x46)    -- BNRY: boundary = PSTART
10. outb(io+0x07, 0xFF)    -- ISR: clear all pending interrupts
11. outb(io+0x0F, 0x3F)    -- IMR: enable all except RST
12. outb(io+0x00, 0x61)    -- CR: page 1, stop
13. outb(io+0x07, 0x46)    -- CURR: Rx write pointer = PSTART
14. Write MAC to PAR0-5 (offsets 0x01-0x06, page 1)
15. outb(io+0x00, 0x22)    -- CR: page 0, start, abort DMA
16. outb(io+0x0C, 0x0C)    -- RCR: accept broadcast + multicast
17. outb(io+0x0D, 0x00)    -- TCR: normal transmit mode
```

### Transmit (Remote DMA Write → NIC memory, then send)

```
1. Set up remote DMA write:
   outb(io+0x08, 0x00)         -- RSAR0: dest addr low (page 0x40 * 256 = 0x4000)
   outb(io+0x09, 0x40)         -- RSAR1: dest addr high
   outb(io+0x0A, len & 0xFF)   -- RBCR0: byte count low
   outb(io+0x0B, len >> 8)     -- RBCR1: byte count high
   outb(io+0x00, 0x12)         -- CR: page 0, start, remote write

2. Write packet data as 16-bit words to port io+0x10:
   for i in 0..len/2: outw(io+0x10, packet_word[i])

3. Wait for RDC (ISR bit 6):
   while !(inb(io+0x07) & 0x40)
   outb(io+0x07, 0x40)         -- clear RDC

4. Initiate transmit:
   outb(io+0x04, 0x40)         -- TPSR: transmit from page 0x40
   outb(io+0x05, len & 0xFF)   -- TBCR0
   outb(io+0x06, len >> 8)     -- TBCR1
   outb(io+0x00, 0x26)         -- CR: page 0, start, TXP
```

### Receive (Read from Rx Ring via Remote DMA)

Each packet in the ring has a 4-byte NIC header:
```
Byte 0: Status (bit 0 = PRX = received OK)
Byte 1: Next packet page
Bytes 2-3: Packet length (little-endian, includes this header)
```

```
1. Check BNRY vs CURR (switch to page 1 to read CURR)
   outb(io+0x00, 0x62)       -- CR: page 1, start
   curr = inb(io+0x07)       -- CURR
   outb(io+0x00, 0x22)       -- CR: page 0, start
   bnry = inb(io+0x03)       -- BNRY
   if (bnry == curr) return   -- ring empty

2. Read 4-byte header via remote DMA:
   outb(io+0x08, 0x00)       -- RSAR0
   outb(io+0x09, bnry)       -- RSAR1 (page address)
   outb(io+0x0A, 0x04)       -- RBCR0: 4 bytes
   outb(io+0x0B, 0x00)       -- RBCR1
   outb(io+0x00, 0x0A)       -- CR: remote read
   status = inw(io+0x10) & 0xFF
   next_page = (inw(io+0x10) >> 8) -- combined read
   ...parse length...

3. Read packet data via remote DMA (skip 4-byte header)

4. Update BNRY to next_page (or next_page - 1 with wrap)
```

### Implementation Notes for Pinecore

1. **ISA card** — no PCI enumeration needed, just probe at known I/O addresses
2. **Detection:** Write to and read back a test register. If echoed, NIC present.
3. **16-bit mode** (WTS=1 in DCR) halves the number of I/O operations
4. **QEMU:** `-net nic,model=ne2k_isa -net user` emulates NE2000 at 0x300, IRQ 9
5. **No bus master DMA** — all transfers are PIO through port 0x10, so no physical memory alignment issues
